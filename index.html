Запуск через файл заблокирован

Браузеры блокируют загрузку ES-модулей, базу данных и API при запуске портала через двойной клик по файлу (протокол file://).

🚀 Как запустить портал правильно:

  1. Зайдите в корневую папку ProgectX в проводнике.
  2. Запустите двойным кликом файл запуск_мультивселенной.bat.
  3. Все проекты автоматически откроются по адресу http://localhost:3000/ и будут работать идеально!

Этот оверлей показывается автоматически при обнаружении прямого открытия HTML-файла.
