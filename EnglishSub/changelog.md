## 1. Текущее состояние / Стек
* **Frontend**: HTML5 (семантическая разметка, блокировка scroll при модалках), Vanilla CSS3 (3 темы: Dark Spotify, Light, Gray Indigo; глассморфизм, `@keyframes` анимации, адаптив), ES6+ JavaScript.
* **Структура файлов**:
  * [index.html](file:///c:/Users/svato/EnglishSub/index.html) — разметка интерфейса, плеера, тренажеров, диалоговых окон.
  * [styles.css](file:///c:/Users/svato/EnglishSub/styles.css) — 2000+ строк, дизайн-система, медиа-запросы, стили тем.
  * [app.js](file:///c:/Users/svato/EnglishSub/app.js) — 3700+ строк, ядро логики, Leitner-интервальное повторение, плеер, чат с ИИ, кэширование, TTS.
  * [essentialWords.js](file:///c:/Users/svato/EnglishSub/essentialWords.js) — Top-1000 частотных слов для тренировок.
  * [playlist_db.js](file:///c:/Users/svato/EnglishSub/playlist_db.js) — CORS-free оффлайн бандл 700+ песен из `Баня.csv`.
* **Интеграции**: Google Gemini API & OpenRouter (модель `google/gemini-2.5-flash`), Web Speech API (синтез речи US/UK, скорость `0.85`), LRCLIB API / Lyrics.ovh (тексты песен).
* **Словарь и Leitner**: Разделенные модальные окна — pocket-инспектор коллекции слов «Мой словарь» и 3D-тренажер «Тренировка слов» (с синхронным переходом). Интервальное повторение (8 уровней, интервалы `[0,1,3,7,14,30,90,180,365]` дней). Карта активности (28 дней), стрик Duolingo-style.
* **Примеры из песен**: Динамический поиск и интеграция реальных строк с переводом из загруженных песен для каждого слова:
  * В **Моем словаре**: Интерактивное раскрытие строки списка (inline drawer) с демонстрацией контекстов, перевода, подсветкой искомого слова (поддерживающей словоформы) и кнопкой удаления.
  * В **Тренажере**: Вывод до 3-х контекстов употребления на обратной стороне 3D карточки для максимального погружения.
* **Премиум Светлая Тема**: Полностью адаптированы и оптимизированы элементы интерфейса **Аудио-диктанта** (текстовые поля ввода, кнопки подсказок, кнопки воспроизведения) и карточка **Культурного кода** для светлой темы, заменяя темные фоны на стильные, легкие, контрастные глассморфические панели.
* **Интерактивный Welcome Dashboard (Главный Экран)**: Разработан полнофункциональный премиум-хаб с глассморфическим сеточным дизайном:
  * **Лого-редирект**: Нажатие на логотип в левом верхнем углу (`#navBrandLogo`) плавно скрывает активные песни и игры и возвращает пользователя на домашний экран.
  * **Виджет статистики**: Вывод количества изученных слов в реальном времени с прямым линком на «Мой словарь».
  * **Динамический список избранного**: Полная интеграция с localStorage — автоматический рендеринг списка любимых треков с мгновенным переходом.
  * **Тренажер-карточки**: Прямые кнопки для запуска «Моего словаря» (`#dashGoDict`) и «Тренировки слов» (`#dashGoTrain`) прямо с главного экрана.
  * **Быстрый старт**: Встроенные карточки Scorpions, Metallica и RHCP для мгновенного знакомства с платформой.
* **Фоновый авто-разбор смысла названия трека**: Интегрировали перевод и объяснение глобального смысла названия трека в общую фоновую prefetch-очередь ИИ:
  * Анализ смысла названия трека (`fetchGeminiSongMeaning`) теперь автоматически ставится самым первым элементом в фоновую очередь перевода при загрузке любой песни.
  * Прогресс-бар фонового перевода (`updatePrefetchUI`) учитывает разбор названия трека как отдельную задачу, увеличивая общее число элементов на +1 и корректно отображая процесс перевода в реальном времени.
* **Резервный AI-перевод названия при отсутствии текста**: Если композиция отсутствует в музыкальных базах или произошла ошибка загрузки текста:
  * Вместо скучного экрана ошибки теперь автоматически рендерится стильная глассморфическая AI-карточка автоперевода названия песни.
  * ИИ в реальном времени (или из локального кэша) переводит название, дает его фонетическую транскрипцию на русском языке, объясняет глубокий смысл трека и делает разбор входящих в название иностранных слов с контекстным переводом!
* **Оптимизация тренажера порядка слов при загрузке**: Устранили баг, когда при запуске мини-игры Phrase Builder во время ожидания AI-разбора в качестве подсказки выводился служебный текст `"Загрузка перевода..."`. Теперь в таких ситуациях автоматически выводится интуитивная инструкция по составлению английской фразы: *"Соберите английское предложение по порядку слов (перевод ИИ загружается...)"*.
* **Интерактивное добавление слов из названия в личный словарь**:
  * Каждое слово в блоке «Разбор слов из названия» (как в боковом сайдбаре стандартных песен, так и в AI-карточке при отсутствии текста) теперь снабжено стильной зеленой кнопкой `➕ В словарь`.
  * При клике слово с переводом мгновенно добавляется в личный словарь («Из песен») для последующей тренировки в интервальном тренажере флеш-карт, а счетчики слов плавно обновляются.
* **Адаптивная контрастность текстовых полей в светлой теме**:
  * Устранили проблему с низким контрастом текста в модальном окне редактирования текста песни (`#editLyricsTextarea`) и поле ручного ввода (`#manualLyricsArea`) при включении светлой темы.
  * Удалили все жестко закодированные темные инлайновые стили фонов (`rgba(0,0,0,0.25)`) и перенесли стилизацию в CSS с полноценной поддержкой темной/светлой тем. В светлой теме поля ввода теперь автоматически приобретают эстетичный светлый фон с высокой читаемостью и контрастным темным текстом.
* **Идеальная прозрачность футера словаря (Glassmorphism)**:
  * Устранили визуальный баг, когда под нижними кнопками управления словаря («Очистить словарь», «Тренировка», «Слово») отображалась сплошная градиентная полоса, которая выглядела как грубая белая плашка. 
  * Заменили полупрозрачный градиент на полностью прозрачный фон (`background: transparent`), благодаря чему кнопки теперь красиво «парят» над общим премиальным стеклянным фоном модального окна на любых темах.
* **Интеллектуальная сегментация текста и фильтрация служебных строк**:
  * Устранили баги с появлением пустых или странных кликабельных строф, состоящих только из служебных меток (например, одиночные строки `(chorus)`, `[Verse 1]` и т.д.). Теперь все служебные маркеры автоматически распознаются и полностью вычищаются из интерактивного списка.
  * Добавили умное разбиение гигантских строф. Если в тексте песни встречается цельный куплет длиной более 5 строк без пустых абзацев, парсер автоматически разделяет его на аккуратные и удобные для изучения кусочки по 3-4 строки.
* **Интерактивный разбор всех песен артиста**:
  * Имя исполнителя в шапке активной песни теперь является кликабельной интерактивной ссылкой с красивой подсветкой при наведении.
  * При клике открывается стильное модальное окно с перечнем всех доступных композиций данного артиста (выборка осуществляется напрямую из глобального массива `csvSongs` загруженных CSV-плейлистов / "бани" и локальных треков с автоматической дедупликацией по названию треков). Клик по любой песне в списке моментально загружает её для разбора и скрывает модальное окно.
* **Переход на лексический подход (Lexical Approach)**:
  * Обновили системный промпт ИИ (Gemini/OpenRouter), разделив возвращаемые учебные данные на два массива: `"vocabulary"` (3-4 ключевых отдельных слова) и `"phrases"` (1-3 устойчивых выражения, словосочетания или фразовых глагола).
  * Полностью переработали интерфейс боковой панели анализа строфы: устаревший скрытый выдвижной ящик и список тегов заменены на два структурированных блока — **📌 Важные слова** и **🎬 Полезные выражения**.
  * Каждый учебный элемент теперь имеет собственную интерактивную Spotify-стилизованную кнопку `➕ В словарь` / `✓ В словаре` прямо в списке, позволяя добавлять или удалять слова/выражения в один клик.
  * Унифицировали базу данных словаря в `localStorage`: добавили свойство `type: "word" | "phrase"`, гарантируя корректное сохранение длинных фразовых выражений целиком в поле `word` с их контекстуальным переводом.
  * Внедрили интеллектуальное авто-масштабирование (font-size) 3D флеш-карт в тренажере. Длинные фразы из 3-4 слов теперь автоматически уменьшаются в размере до `1.45rem` или `1.15rem` (вместо фиксированных `2rem`), полностью предотвращая выход текста за пределы карточки.
* **Геймификация словаря — Режим «Match» (Найди пару на время)**:
  * Внедрили интерактивный игровой режим «Match», вдохновленный Quizlet, для эффективного игрового закрепления изученных слов и выражений.
  * Разместили кнопку запуска `🎮 Match` рядом с поисковой строкой в окне Словаря. При запуске обычный список скрывается, уступая место динамической игровой зоне с крупным высокоточным секундомером (с точностью до 0.1 сек).
  * Игра автоматически отбирает 6 случайных слов/выражений из личного словаря пользователя (требуется не менее 6 слов для запуска) и генерирует сетку из 12 перемешанных плиток (Fisher-Yates shuffle).
  * Реализовали интуитивный геймплей: при выделении плитки подсвечиваются мягким Spotify-зеленым свечением. При верном сопоставлении плитки плавно растворяются, а при ошибке окрашиваются в красный оттенок и совершают быструю анимацию встряхивания (shake), сбрасывая выделение через 0.5с.
  * По завершении (все 12 плиток сопоставлены) таймер останавливается и выводится красивое праздничное поздравление с точным временем прохождения игры. При закрытии модального окна или клике на выход игра безопасно завершается и сбрасывает все таймеры.
* **Геймификация словаря — Режим «Learn» (Тест с 4 вариантами ответа)**:
  * Внедрили интерактивный игровой режим тестирования «Learn», вдохновленный Quizlet, для эффективного запоминания и проверки изученных выражений.
  * Разместили кнопку запуска `📝 Тест` рядом с кнопкой `🎮 Match` над списком слов. При клике основная область скрывается, уступая место интерактивному полигону тестирования.
  * Реализовали умный алгоритм генерации вопросов: раунд формируется из 10 случайных слов словаря (требуется не менее 4 слов для запуска). Для каждого вопроса ИИ отбирает целевое английское слово и 3 случайных перевода (дистрактора), перемешивая их с помощью алгоритма Фишера-Йетса.
  * Интегрировали высокоточную систему обратной связи:
    * При правильном выборе кнопка ответа подсвечивается зеленым неоновым свечением, прогресс-бар плавно заполняется, и через 1 секунду загружается следующий вопрос.
    * При неверном выборе нажатая кнопка окрашивается в красный со встряской (shake), правильная альтернатива принудительно выделяется зеленым для облегчения запоминания, а задержка увеличивается до 2 секунд.
    * В реальном времени вычисляется и отображается текущий показатель точности (Accuracy) прохождения раунда.
  * Разработали стильный финальный экран результатов (с подсчетом процента точности и правильных ответов) и кнопкой возврата к обычному словарю.
* **Интерактивный модуль «AI Roleplay / Симулятор общения»**:
  * Внедрили продвинутый симулятор диалога с ИИ для вывода изучаемой лексики в активный разговорный запас.
  * Разместили интерактивную кнопку `💬 Практика` в развернутом drawer-окне каждого слова/выражения в списке словаря, а также круглую кнопку `💬` на лицевой стороне 3D-карточек в Leitner-тренажере.
  * Создали компактное всплывающее окно чата в стиле Telegram/iMessage с поддержкой плавной анимации пузырей сообщений, индикатором печати ИИ (мигающие точки) и адаптивностью под светлую и темную темы.
  * Реализовали умную логику ведения диалога:
    * При старте ИИ (через сохраненный ключ OpenRouter/Gemini) генерирует неформальную жизненную ситуацию и задает наводящий вопрос, подталкивая пользователя обязательно использовать целевую фразу.
    * Ответ пользователя отправляется обратно ИИ, который мягко исправляет грамматические неточности, оценивает естественность употребления фразы и продолжает диалог по смыслу.
    * После 3-х кругов общения активируется кнопка «Завершить практику», при клике на которую начисляется **+15 очков активности** в карту активности словаря.
* **Интерактивный Видеокурс English Galaxy (Английский по видео)**:
  * Внедрили интерактивный Lesson Tracker для курса Александра Бекишева *"Английский от нуля до продвинутого. Практический курс по приложению English Galaxy"* (Уровень А0, 50 уроков).

  * Разработали премиальное glassmorphic модальное окно (`#videoCourseModal`) с двухколоночным адаптивным лейаутом, фильтрацией уроков (Все / Непройденные / Пройденные), мгновенным текстовым поиском по номеру/названию и встроенным iframe-плеером YouTube.
  * Интегрировали автоматическую синхронизацию прогресса (localStorage) и геймифицированную систему активности: первое прохождение любого видеоурока начисляет **+10 очков активности** через `recordActivity(10)`, отражаясь на тепловой карте словаря, и сопровождается лаконичным всплывающим тост-уведомлением.
  * Оптимизировали контрастность в светлой теме (Light Theme): принудительно зафиксировали белый цвет для текста плейсхолдера поверх всегда темного экрана плеера и применили высококонтрастный насыщенный фиолетовый (#7c3aed) для активных кнопок фильтров, очков активности и бейджей уроков.
* **Отдельное поле «Значение / Толкование» для слов и фраз**:
  * Добавили полноценную поддержку детального толкования (значения, идиоматического смысла или примеров контекста) отдельно от перевода.
  * Форма ручного добавления слова/фразы в словарь расширена многострочным текстовым полем `Значение / Толкование (опционально)` с адаптивным плейсхолдером.
  * Создали визуальный блок значения в карточках словаря: если у слова заполнено значение, оно отображается изящным наклонным шрифтом с левым фиолетовым отступом-полоской под основным переводом.
  * Интегрировали редактирование толкования в инлайн-форму редактирования карточки непосредственно в списке.
  * Реализовали фоновую инициализацию поля `definition: ""` для всех прошлых словарных записей для гарантированной обратной совместимости.
* **Двухуровневая система организации словаря (Папки + Пользовательские Категории)**:
  * Внедрили полноценное конкурентное разделение всех слов и фраз на физические **Папки (Коллекции)** и смысловые **Категории (Теги)**.
  * Разработали современный двухколоночный адаптивный фильтр в модальном окне Словаря: фильтр по папкам (`📁 Папка`) и фильтр по категориям (`🏷️ Категория`).
  * Реализовали возможность динамического создания собственных категорий на лету: клик по фиолетовому плюсу (`addCustomCustomCategoryBtn`) запрашивает имя новой категории, валидирует её и мгновенно добавляет во все списки выбора.
  * Обновили интерфейс карточек слов в списке: теперь на каждой карточке отображаются стильные inline-теги с иконками папки (`📁`) и категории (`🏷️`), обеспечивая великолепную визуальную считываемость структуры.
  * Расширили форму ручного добавления слова/фразы (`manualWordCustomCategory`) и форму инлайн-редактирования непосредственно в списке словаря — теперь пользователь может свободно менять папку и категорию для любого сохраненного элемента (слово или фраза).
  * Интегрировали автоматическую обратную миграцию: при запуске все ранее сохраненные слова без категории автоматически помечаются тегом `'Без категории'`.
* **Интеграция кнопки «Добавить фразу» в ручной режим словаря**:
  * Полностью переработали нефункционирующую кнопку `+ Фразу` (`#addPhraseBtn`) в словаре. Теперь она работает абсолютно аналогично ручному добавлению слов.
  * При вводе английской фразы (длиной от 3-х слов) и нажатии `+ Фразу` автоматически открывается то же премиальное модальное окно ручного ввода, динамически адаптируясь: заголовок меняется на *"➕ Добавить фразу вручную"*, текстовые метки и плейсхолдеры адаптируются под режим ввода фразы, а поле английского текста заполняется введенным текстом.
  * При сохранении фразы в `personalDictionary` ей автоматически присваивается признак `type: "phrase"`, что гарантирует её корректную фильтрацию и отображение в разделе "Фразы" словаря.
* **Интерактивный модуль «Справочник правил» ( Grammar Rules Handbook )**:
  * Разработали и внедрили полноценную личную базу знаний грамматических правил (`#rulesModal`) в фирменном глассморфическом стиле.
  * Точка входа: добавили в верхнее меню (Navbar) стильную кнопку `📐 Правила` (`#openRulesBtn`) небесно-голубого цвета с тенью и эффектом наведения.
  * Двухпанельный интерфейс:
    * **Список правил (Режим чтения)**: визуализирован в виде премиального аккордеона с анимированной стрелкой шеврона. Позволяет просматривать правила по клику на заголовок, сдвигая и раскрывая контент с сохранением форматирования абзацев и переносов строк. Включает кнопку удаления правил `🗑️`.
    * **Добавить правило (Режим создания)**: удобная форма с валидацией полей ввода названия и содержимого, предфокусировкой, а также автоматическим переключением вкладок при успешном сохранении.
  * Хранение данных: автоматическая синхронизация с `localStorage` (ключ `grammar_rules`), поддержка темной/светлой/серой тем (высокий контраст полей ввода в светлой теме) и начисление **+5 очков активности** на тепловую карту при создании нового правила.
* **Сверхнадежный 6-уровневый движок поиска текстов песен ( Genius + AI + Fuzzy LRCLIB )**:
  * Полностью переработали механизм автоматического поиска текстов песен, заменив простой последовательный опрос двух баз данных на интеллектуальную, отказоустойчивую 6-уровневую систему.
  * Уровни поиска:
    1. **LRCLIB Exact Lookup**: Прямой поиск точной сигнатуры метаданных.
    2. **LRCLIB Fuzzy Signature Search**: Нечеткий сигнатурный поиск (по полям `track_name` и `artist_name`) для устранения влияния мелких опечаток, дефисов и скобок.
    3. **LRCLIB Fuzzy Query Search**: Полнотекстовый нечеткий поиск по строке запроса `artist + title` через параметр `q`.
    4. **Lyrics.ovh**: Быстрый классический безналоговый fallback-провайдер текстов.
    5. **Интеграция Genius Scraper**: Реализовали полноценный клиентский парсер Genius.com, который обходит CORS-ограничения через прокси-сервис `allorigins.win`, находит правильный URL страницы песни с помощью Genius Search API, загружает её HTML и извлекает чистый текст песни из контейнеров `div[data-lyrics-container="true"]` с сохранением переносов строк.
    6. **Gemini AI Lyrics Fallback**: Задействовали искусственный интеллект в качестве ультимативного источника. Если все базы и парсеры вернули ошибку, приложение делает запрос к API-ключу пользователя (Gemini 2.0 Flash или OpenRouter) с требованием сгенерировать аутентичный, чистый текст песни на английском языке.
* **Поиск примеров в названиях песен ( Song Title Example Fallback )**:
  * Расширили алгоритм `findSongExamples(word)` фазой 2. Если искомое слово/выражение отсутствует в текстах песен, движок сканирует названия песен.
  * При совпадении формируется красивый блок с пометкой `[Упоминание в названии] {Song Title}` и указанием исполнителя, что предотвращает появление пустых блоков для редких слов, вынесенных авторами в заголовки.

## 4. Ключевые правила проекта
1. **Только Vanilla CSS**: Никаких сторонних UI-фреймворков (Tailwind/Bootstrap). Применение CSS-переменных (`:root`, `body.light-theme`, `body.gray-theme`).
2. **CORS-Free по умолчанию**: Поддержка оффлайн запуска (`file://`). Читать локальный бандл `playlist_db.js` перед сетевыми fetch.
3. **Безопасность API**: Запросы к ключам проводить строго через `getAPIKey()` (проверка `localStorage.getItem('user_api_key')` перед fallback-ключом).
4. **Сохранение комментариев**: Оставлять нетронутыми существующие JSDoc/комментарии при обновлении JS-файлов.
5. **GPU Оптимизация**: Использовать `will-change` для 60 FPS, уважать пользовательский флаг `.no-blur-mode` (отключение `backdrop-filter`).
